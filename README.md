# DH&MM — Medieval Museum (Rooms 18–19) — Project Skeleton

This is the repository of the end-of-the-course project for the integrated course in "Digital Heritage and Multimedia", a.y. 2023/2024, DHDK, Univerisy of Bologna

## Structure
- `index.html` — main site that presents the project
- `assets/css/style.css` — minimal styling
- `assets/js/main.js` — small JS helpers
- `twine/story-placeholder.html` — iframe target
- `twine/story.twee` — Twee starter
- `docs/DESIGN-BRIEF-template.pdf` — drop-in template
- `docs/Design-Brief—draft.md` — Markdown draft aligned with template
